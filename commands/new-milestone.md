<objective>
Start a new milestone cycle in the current project.
</objective>

<process>
1. Read `.planning/ROADMAP.md` and `.planning/STATE.md` to understand current project state.
2. Identify the next milestone scope based on roadmap and completed phases.
3. Create milestone definition with clear goals, phases, and success criteria.
4. Update `.planning/ROADMAP.md` with the new milestone.
5. Update `.planning/STATE.md` to reflect the new active milestone.
6. Report the milestone scope and first phase to work on.
</process>
