<objective>
Plan additional phases to address gaps identified during milestone audit.
</objective>

<process>
1. Read the milestone audit results.
2. For each identified gap, create or update a phase to address it.
3. Update `.planning/ROADMAP.md` with any new phases.
4. Report the gap-filling plan and suggest execution order.
</process>
