<objective>
Analyze the codebase structure, dependencies, and patterns.
</objective>

<process>
1. Scan the project structure (directories, key files).
2. Identify architecture patterns, frameworks, and tech stack.
3. Map key dependencies and their relationships.
4. Document findings for use in phase planning.
5. Report the codebase overview.
</process>
